# Starting in Dev

## Spinning Up

```shell script
$> {your preferred invokation to make python env}
$> pip install -r requirements.txt
$> yarn install
```

## Running .NET

```powershell
$> dotnet restore
$> dotnet run
```

## Running Django

```shell script
$> python manage.py runserver
```

## Running Webpack in Dev

```shell script
$> yarn start
```
